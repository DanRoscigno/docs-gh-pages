skewPop

计算给定序列的 [偏度] (https://en.wikipedia.org/wiki/Skewness)。

语法

skewPop(expr)

参数

expr — 表达式 返回一个数字。

返回值

给定分布的偏度。类型 — Float64

示例

SELECT skewPop(value) FROM series_with_value_column;
